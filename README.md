# Sales Performance Dashboard

## Project Overview
An interactive Excel dashboard built using the Superstore dataset to analyze sales performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Dashboard Features
- Monthly Sales Trend
- Sales by Region
- Sales by Category
- Profit Analysis
- Top Products
- Interactive Filters

## Dataset
Sample Superstore Dataset

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Dashboard Design
- Excel Functions
- Data Visualization
